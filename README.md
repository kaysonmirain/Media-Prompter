# Media Prompter

Upload an image or video and get back a prompt, captions, detected objects, scene info, mood, and tags.

## Run it

```bash
chmod +x start.sh
./start.sh
```

Opens at http://localhost:6666. First run installs dependencies and downloads model weights — takes a few minutes.

Uses BLIP, YOLOv8x, CLIP, and EfficientNet under the hood.
